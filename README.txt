


RUN THE PROGRAM
1. cd to the project directory
2. type "make" to compile
3. type "./driver" to run

USE THE PROGRAM
1. You will be prompted to input the name of the arff file to test.
You can use any of these test files already included:
table3_10_fg.arff
weatherData.arff
wilkensonMatrix.arff

2. You will be given a list of all of the attributes within the file, and asked
to select the decision attributes. 
Select the decision attributes by inputting a list of numbers given separated by spaces

3. The list of coverings will be displayed.
